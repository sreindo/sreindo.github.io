ES6初步

发表于 2020-05-17

ES6初步

一、ECMAScript与JavaScript

ECMAScript：标准

JavaScript：实现

1995-》 1997（第一个版本） -》1999（ES3）-》2009（ES5）

2015（ES6） -》 2016(ES7)

现代浏览器都基本上支持ES6

二、变量与常量

ES5：var全局作用域和(函数中)局部作用域

ES6：let 声明一个块级作用域的变量，块：{ }

const 声明一个常量，必须在声明时赋值

模板字符串 用反引号（`）定界的字符串，可以在模板字符串中用${变量}来来引用变量的值

三、解构

解构技术：将数组、对象等分解成组成此数据结构的技术

四、函数

  • 默认参数

  • 剩余参数（rest参数）

    …vrest 由小数点+变量构成，将剩余所有参数放在这个数组中。

  • 箭头函数

​ 原匿名函数是：var fun = function(…){…}

​ 可写成 var fun = (…) => {…}

​ 箭头函数基本语法：

​ (param1, param2, …, paramN) => {statements}

//如果只有一个参数，可以不写圆括号；singleParam => {expression}

//如果没有参数，一定要写圆括号 ()=>{expression}

//如果函数体语句只有返回一个值的语句，可以不写{}和return

singleParam => expression 相当于(singleParam) => {return expression;}

高级语法

//加括号的函数体返回字面量表达式

params => ({foo : bar}) //目的是返回对象{foo : bar}

//支持剩余参数和默认参数

五、类

在ES6中，提供了class关键字，用于定义类

并提供了extends来实现类的继承

六、模块(module)

每一个独立的ja文件都是一个独立的作用域，是独立存在、模块间互不影响。

在大型的项目中可能会有很多js文件，并且不同的js文件还会互相引用。

用来分解、封装和组织代码的工具。

  • export 和 import

export 用来到处当前文件（模块）中的变量函数、类等

import 用来导入其他模块间的变量，函数。类等内容

单独的导出导入

批量导出导入

在html中使用模块，需要在JavaScript中标记中使用 type=”module”

事件监听器

不同事件句柄，只能绑定一次，只能插接一个事件处理函数

对于事件监听器，一次可以插接多个

IE8标准

DOM标准下的事件监听器

支持W3C标准事件监听器（IE9 Chrome FireFox…)

addEventListener设置监听器

addEventListener(type, handler, boolean)

接收三个参数：

  1. 第一个参数：事件类型名(click, )

  2. 第二个参数：回调处理函数（监听器函数）

  3. 第三个参数：注明该处理函数是在事件传递过程中的捕获阶段（true）被调用

    ​ 还是在冒泡阶段（false）被调用

removeEventListener取消监听器

跨浏览器注册与移除元素事件监听器方案

事件处理程序的返回值

  • 事件处理程序的返回值
    在JavaScript中通常事件处理程序不需要有返回值，这时浏览器会按默认方式进行处理；但很多情况下需要使用返回值，来判断事件处理程序是否正确进行处理。
  • 返回值类型：boolean值
    浏览器根据返回值的类型决定下一步如何操作。当返回值为true，进行默认操作； 当返回值为 false，阻止浏览器的下一步操作。
  • 基本语法：事件句柄=“return 函数名（参数）;”

第一种：return boolean；

第二种：return function（）；
